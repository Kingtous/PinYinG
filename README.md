# PinYinG - 拼音猜测

> 受Wordle启发，拼音版本的“PinYinG?”现已推出。

制作过程参考了国内其他人做的拼音版本的wordle，如猜成语等。

但发现其他版本总体难度偏大，没有相当的词语积累，很难完赛。

本版本旨在降低游玩难度，限定在猜测两字词语，从而降低游玩难度，增加乐趣。
